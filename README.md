# Демонтажные работы — Мир Тимошенко

Сайт-визитка по демонтажным работам.

## Текущее состояние

`index.html` — одностраничный лендинг (чистый HTML + CSS, без сборки).

Секции:
- **Hero** — «Сносим стены. Оставляем порядок.»
- **Обо мне** — Мир Станиславович Тимошенко
- **Связаться** — форма заявки (имя, телефон, задача)
- **Слоган**
- **Сильные стороны** — «Почему мне доверяют»
- **Подвал**

Стек: HTML, CSS (переменные в oklch), шрифты Inter + Oswald, иконки Lucide.

## Планы

- Перевести на Next.js (роутинг, компоненты, оптимизация изображений)
- Отдельные страницы под услуги (демонтаж стен, полов, сантехники, вывоз мусора)
- Портфолио выполненных работ
- Рабочая форма заявки (отправка на почту / в Telegram)
- SEO: отдельные title/description на страницу, sitemap, robots
